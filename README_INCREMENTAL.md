# Инкрементальный поиск Puzzle 71

## Стратегия

Постепенное приближение к целевому адресу `1PWo3JeB9jrGwfHDNpdGK54CRas7fsVzXU` через короткие запуски и анализ результатов.

## Использование

### 1. Простой инкрементальный поиск

```bash
./incremental_search.sh
```

Запускает один поиск с текущим паттерном `1PWo3Je*zXU` и анализирует результаты.

### 2. Автоматический инкрементальный поиск

```bash
./incremental_search_auto.sh
```

Автоматически:
- Запускает несколько итераций поиска
- Анализирует найденные адреса
- Автоматически увеличивает префикс/суффикс на основе результатов
- Продолжает до достижения цели или максимума итераций

### 3. Анализ результатов вручную

```bash
python3 analyze_found_addresses.py incremental_results.txt 1PWo3JeB9jrGwfHDNpdGK54CRas7fsVzXU
```

## Параметры

В скриптах можно изменить:
- `SEARCH_TIME` - время одного поиска (по умолчанию 60-120 сек)
- `CURRENT_PREFIX` - начальный префикс (по умолчанию "1PWo3Je")
- `CURRENT_SUFFIX` - начальный суффикс (по умолчанию "zXU")
- `MAX_ITERATIONS` - максимальное количество итераций (только для auto)

## Пример работы

1. **Итерация 1**: Поиск `1PWo3Je*zXU`
   - Найдено: `1PWo3JeB9j*...` (9 символов совпадения)
   - Следующий шаг: `1PWo3JeB*zXU`

2. **Итерация 2**: Поиск `1PWo3JeB*zXU`
   - Найдено: `1PWo3JeB9jr*...` (11 символов совпадения)
   - Следующий шаг: `1PWo3JeB9*zXU`

3. **Итерация 3**: Поиск `1PWo3JeB9*zXU`
   - И так далее...

## Рекомендации

1. Начните с коротких запусков (60-120 сек) для быстрой оценки
2. Если находите много совпадений - увеличьте время поиска
3. Если совпадений мало - проверьте правильность паттерна
4. Используйте прогресс файл для возобновления поиска
